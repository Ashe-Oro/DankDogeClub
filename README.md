# Dank Doge Club

Static export of [dankdogeclub.com](https://dankdogeclub.com) — a flat,
self-contained copy of the site for hosting on any static host.

## Pages

- Homepage
- All Series, plus subcategory pages: Life of Doge NFT Comic, General
  Series, Series #1, Series #0, OG Rare Doge Club NFTs, The Doge House

## Hosting

Serve the repository root with any static file server. No build step or
server-side code is required.
